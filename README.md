## Запуск
В корне проекта:

```bash
git submodule update --init --recursive
mkdir build
cd build
cmake ..
make all
```

Дальше в зависимости от цели:

- Запуск с ручным вводом:
    ```bash
    ./src/solution
    ```

- Запуск тестов:
    ```bash
    ./tests/solution_test
    ```
